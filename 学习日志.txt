原型 每声明一个function 都会有一个prototype原型 prototype原型是函数的一个默认属性 在函数的过程中会自动添加
原型链 原型链是实现继承的主要方法 
思想：父类的实例 作为子类的原型 
function Cat(){ 
}
Cat.prototype = new Animal();
Cat.prototype.name = 'cat';

原型链继承
构造继承 
核心：使用父类的构造函数来增强子类实例，等于是复制父类的实例属性给子类（没用到原型）
function Cat(name){
  Animal.call(this);
  this.name = name || 'Tom';
}
实例继承
核心：为父类实例添加新特性，作为子类实例返回
function Cat(name){
  var instance = new Animal();
  instance.name = name || 'Tom';
  return instance;
}
function Cat(name){
  var animal = new Animal();
  for(var p in animal){
    Cat.prototype[p] = animal[p];
  }
  Cat.prototype.name = name || 'Tom';
}
组合继承
核心：通过调用父类构造，继承父类的属性并保留传参的优点，然后通过将父类实例作为子类原型，实现函数复用

function Cat(name){
  Animal.call(this);
  this.name = name || 'Tom';
}
Cat.prototype = new Animal();
